# data viewer

## Tech

- Vue 2
- Vuetify
- Vuex
- Vue-router
- Axios
- json-server
- html2canvas

### Project setup

```
npm install
```

### Compiles and hot-reloads for development

first load up the fake DB using in terminal:

```
json-server --watch .\db\db.json
```

than run the program in different terminal using the comment:

```
npm run serve
```
